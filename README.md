## Installation

```bash

python -m venv venv
source venv/bin/activate  # Linux/macOS
venv\Scripts\activate     # Windows
```

## Requirements

- Python 3.10 or higher
- Webcam for gesture capture
- CUDA-compatible GPU (optional, for faster training)
